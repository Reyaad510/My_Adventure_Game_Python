# My_Adventure_Game_Python

Making an adventure game using python that a user plays from their terminal.

- So far I have created a simple combat system allowing user to attack, use skills, use items(doesn't skip a turn yet), and after a certain condition will have access to an ultimate move.
- Created a set of characters and some opening dialogue scenes with main character Zidane and his crew.
- User able to equip weapons or armor.
- Created an inventory system that holds weapons, armor, and items that heal or restore MP.
